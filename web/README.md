# Cordel Creator Web

Interface React para criar histórias de cordel a partir de um briefing simples.

## Rodando

```bash
npm install
npm start
```

Por padrão, o frontend chama `http://localhost:5555/api/cordel`.
Para trocar a URL da API, crie `.env.local`:

```env
REACT_APP_API_URL=http://localhost:5555
```
